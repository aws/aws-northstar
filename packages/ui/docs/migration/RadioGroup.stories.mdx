# Migrating from NorthStar legacy - RadioGroup

Use <Link href='https://cloudscape.design/components/radio-group' target='_blank'>Cloudscape RadioGroup component</Link>

If the RadioGroup is used in a Form setting, it is recommended to 
use <Link href='https://aws.github.io/aws-northstar/?path=/story/components-formrenderer--default' target='_blank'>NorthStar v2 FormRenderer component</Link> 
to build the form from JSON schema.

## Update Import

```diff
- import RadioGroup from 'aws-northstar/components/RadioGroup';
+ import RadioGroup from '@cloudscape-design/components/radio-group';
```
