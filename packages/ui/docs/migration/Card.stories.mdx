# Migrating from NorthStar legacy - Card

Cloudscape does not expose an interface for individual Card component. 
Use <Link href='https://cloudscape.design/components/cards' target='_blank'>Cloudscape Cards component</Link> to display a list of card. 

For use cases where only single card is used, use <Link href='https://cloudscape.design/components/container' target='_blank'>Cloudscape Container component</Link>
to display the content of a Card.
