# Migrating from NorthStar legacy - TimePicker

There is not a matched TimePicker component avaliable on Cloudscape. However, 
you can use <Link href='https://cloudscape.design/components/time-input' target='_blank' external>Cloudscape TimeInput component</Link> to allow users enter an absolute time value.

## Update Import

```diff
- import TimePicker from 'aws-northstar/components/TimePicker';
+ import TimeInput from '@cloudscape-design/components/time-input';
```
