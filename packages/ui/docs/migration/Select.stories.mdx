# Migrating from NorthStar legacy - Select

Use <Link href='https://cloudscape.design/components/select' target='_blank'>Cloudscape Select component</Link>

If the Select is used in a Form setting, it is recommended to 
use <Link href='https://aws.github.io/aws-northstar/?path=/story/components-formrenderer--default' target='_blank'>NorthStar v2 FormRenderer component</Link> 
to build the form from JSON schema.

## Update Import

```diff
- import Select from 'aws-northstar/components/Select';
+ import Select from '@cloudscape-design/components/select';
```
